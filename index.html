CRM Web App - Web Development

pla-crm-demo.web.app.com

This is a CRM (Customer Relationship Manager) App built for Plasencia Cigars. They currently use it to manage thier accounts. The app contains information about the accounts such as invoices, sales info, contact info. It also has the ability for users to submit orders for these acccounts, as well as notes so other sales representative can follow up and be aweare of the current status of an account. This app also communicates with Quickbooks accounting software and transfers data automatically throgh the day.

It was built with React and the Firebase Suite (Firestore, Cloud Functions & Google Auth)
